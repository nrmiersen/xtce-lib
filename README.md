Library of XTCE tools such as:
- A unified XTCE model that is a superset of all XTCE versions. It is simplified to align with Python standards rather than XML standards, removing redundant intermediate classes such as ParameterSetType. It will have functionality to:
  - Be generated by existing an existing XTCE file.
  - Serialize to any XTCE version.
  - Perform semantic validation to verify the contents of the XTCE make logical sense.
- A class to represent and interact with XTCE object paths.
- A class to interact with existing XTCE files, with functionality for things like parsing the XTCE into a Python object and performing XML schema validation.

Supports XTCE versions [1.1](https://www.omg.org/spec/XTCE/1.1/), [1.2](https://www.omg.org/spec/XTCE/1.2/) and [1.3](https://www.omg.org/spec/XTCE/1.3/)

**Alpha Software:** This package is currently in the alpha stage of development. The API is not stable and is subject to breaking changes in future releases. Please use with caution and report any bugs or issues you find.
